# fullstack-webshop

## Miért?

Az alkalmazás célja, hogy jelentős előnyt biztosítson a vállalkozásoknak a piaci jelenlét, az ügyfélszolgálat és a hatékonyság tekintetében, miközben biztonságos, gyors és kényelmes vásárlási élményt teremt az ügyfelek számára.

### Features

Adminisztrátori nézet

- Felhasználók kezelése
- Terméklista frissítése
- Kérelmek kezelése

Támogatás

- OAuth bejelentkezés
- Stripe fizetés
- API kérések

## Hátrányok

Az Angular alkalmazások néha több sablonkódot tartalmaznak más keretrendszerekhez képest, ami nagyobb kódbázist eredményez, amelyet nehezebb lehet karbantartani.
